# Prep Prompt

This is not a skill — it's a prompt participants paste into Claude Chat before GitHub is connected.

Send this in the welcome email a few days before the workshop.

---

## The Prompt

Copy and paste this entire block into Claude:

```
I'm preparing for a BuildFirst workshop where I'll learn to build AI tools. I need to:

1. Make sure I have Claude Pro ($20/month) — I should already be using it if I'm talking to you
2. Install Claude Code — this is a terminal tool for building more sophisticated things
3. Create a GitHub account (free) — this is where I'll access the workshop materials
4. Download Wispr Flow — this lets me talk instead of type

I'm an absolute beginner. I've never used a terminal or GitHub. Walk me through each step one at a time. Be patient, be clear, and tell me exactly what to click and what to paste. When something looks intimidating, tell me it's simpler than it looks.

Let's start with step 1.
```

---

## Purpose

- Guide them through setup (Claude Pro, Claude Code, GitHub, Wispr Flow)
- First real interaction with Claude — sets the tone
- Shows Claude can help them do unfamiliar things
