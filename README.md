# BuildFirst Skills

A set of guided skills that help absolute beginners build their first AI-powered tool in a 4-hour workshop.

## What This Is

BuildFirst is a hands-on workshop where you build something real with Claude's help. These skills guide you through each phase — from your first quick win to leaving with a finished project and ideas for your next one.

**The skills are training wheels.** The real lesson is that you can talk to Claude about anything. Once you've been through the workshop, you won't need the skills anymore.

## The Workshop Arc

| Phase | What Happens |
|-------|--------------|
| **Quick Win** | Build something working in 30 minutes |
| **Dream Capture** | Share your messy ideas without judgment |
| **PRD Builder** | Shape ideas into a clear plan |
| **Build** | Build it step by step, testing as you go |
| **Use** | Try it for real |
| **Refine** | Fix what matters, save the rest for later |
| **Retrospective** | Discover what you actually learned |
| **Next Project** | Leave with a prompt for your next build |

## Before the Workshop

A few days before, you'll receive a **Prep Prompt** to paste into Claude. It walks you through setting up:

1. Claude Pro ($20/month)
2. Claude Code ($20/month)
3. GitHub account (free)
4. Wispr Flow (for voice input)

You don't need to know how to use any of these yet. Claude will walk you through it.

## How to Use the Skills

During the workshop, your facilitator will tell you which skill to use at each phase.

### If you're using Claude Code:

1. Clone this repository:
   ```
   git clone https://github.com/[your-org]/buildfirst-skills.git
   ```

2. The skills will be available to Claude Code automatically

### If you're using Claude Chat:

1. Open the skill file you need (in the `skills/` folder)
2. Copy the entire contents
3. Paste it into Claude as a system prompt or at the start of your conversation

## Templates

The `templates/` folder has blank documents you can copy and fill in:

- **PRD Template**: Structure for defining what you're building
- **Parking Lot**: A place to save ideas and improvements for later
- **Glossary**: Your personal dictionary of tech terms, tied to what you built

## Getting Help

During the workshop, just ask Claude. That's the whole point.

After the workshop, you can:
- Come back to these skills if you get stuck
- But try without them first — you know more than you think

## The Philosophy

- Build small, test often
- Done is better than perfect
- You're capable; you're just new to this
- The skills are temporary; the confidence is permanent
